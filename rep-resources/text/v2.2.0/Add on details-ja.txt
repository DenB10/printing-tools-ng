PrintingTools NG で提供される主な拡張機能は次のとおりです。
その他の機能については設定内容を参照してください。

<b>設定 UI、PrintinTools NG ツールバーボタンとメニュー：</b>

- 25か国語にローカライズされた新しい設定 UI になりました。
- ヘルプを刷新しました。
- ツール/アドオンとテーマの拡張機能の管理、またはツールバーボタンから [PrintingTools NG 設定] にアクセスできます。
- ファイルメニューに [PrintingTools NG 設定] を追加しました。
- ツールバーボタンに [印刷]、[印刷プレビュー]、[PrintingTools NG 設定] [ヘルプ] を用意しました。

<b>メッセージのレイアウトとフォーマット：</b>

- CSS でカスタマイズ可能なヘッダー境界線を追加します。
- ヘッダーの背景色を追加します。
- ユーザー名、フォルダ名、または任意のタイトルを追加します。
- フルカスタマイズ可能な日時を含む複数の日時形式に対応します。
- 標準または拡張ヘッダーを非表示にします。
- 標準ヘッダーを並び替えます。
- アイコン付き添付ファイルリストをヘッダーに追加します。
- 本文最下部の添付ファイルリストを非表示にします。
- デジタル署名ファイルを表示します。
- インライン添付ファイルを非表示にします。
- フォントファミリーとサイズを指定できます。
- 引用テキストの色、フォント、サイズを強調表示します。
- ヘッダー付きで複数の選択したテキストのみを印刷します。
- スレッドを含む複数メッセージを印刷します。
- FiltaQuila と連携してサイレントで自動印刷します。 (FiltaQuila 3.4+)

<b>PDF 出力：</b>

- PDF 印刷で自動命名で保存します。
- 高度な設定を可能にするトークンベースの命名システムがあります。
- 出力フォルダーを設定したサイレント印刷ができます。
- ASCII ファイル名の文字と絵文字をフィルタリングします。

<b>プリンターとページ設定：</b>

- サイレント印刷が設定できます。
- 恒久的なページ設定を保存します。
- 倍率、ページ指定、余白、ヘッダーとフッターをカスタム可能にします。
- ページ指定で最初のページのみの印刷ができます。

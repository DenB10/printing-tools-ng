PrintingTools NG offers many options to format the header section when printing.
This screenshot shows several commonly used options (all are configurable):

- A different header order than the standard Thunderbird order
- Alignment of the headers 
- The date in a custom format including the GMT offset in brackets
- A custom border, top and bottom only defined with a a simple CSS statement :
		border-top: magenta 2px; border-bottom: magenta 2px;
- A background color for the headers 
- A page header with the username and message folder
- Inclusion of an attachment list in the headers block with file type icons
- Hiding the standard Thunderbird attachment list added to the message body

Message Printing Tools (NextGen)

New update for Thunderbird v91
New toolbar buttons and formatting options

---------------------------------------

About this Add-on
PrintingTools NG provides numerous layout and formatting options for
printing messages (the address book support is temporarily disabled)

Version 2.1.0 : v91 Update - March 4, 2022

- Allow compatibility for v91.*
- New mail toolbar button and mail header button
- Integration with FiltaQuila for automatic printing
- New option for hiding the body attachment list
- New debug options
- Address book capability temporarily disabled


The following are the main enhancements provided by PrintingTools NG,
see the options for additional features.

Options & Menus:
- Options access from the Tools/Add-on options.
- Printing Options menu added to File menus.
- Toolbar buttons for print, preview and options

Message Layout & Formatting:
- Add customizable header border with CSS.
- Add header background color.
- Add Username, Folder or custom title.
- Several date formats
- Hide standard or extended headers.
- Reorder standard headers.
- Add the attachment table to header with icons.
- Hide the in body attachment list
- Show signature files.
- Hide inline attachments.
- Configure font family and size.
- Highlight quoted text color, font and size.
- Print selected text only.
- Print multiple messages
- Automatic, silent printing with FiltaQuila
Requires FiltaQuila v3.4 (coming shortly)
Beta available :
https://github.com/RealRaven2000/FiltaQuilla/issues/152#issuecomment-1050708632

メッセージ印刷ツール （NextGen）

Thunderbird 91 に対応しました。
ツールバーのボタンと書式設定に各種の追加があります。

---------------------------------------

このアドオンについて
PrintingTools NG は、メッセージの印刷に多くのレイアウトや書式設定を提供します。
(アドレス帳のサポートは一時的に無効になっています)

バージョン2.1.0 : Thunderbird 91 対応アップデート - 2022年3月4日

- Thunderbird 91.* に対応しました。
- ツールバー用ボタンとメッセージヘッダー用ボタンを追加しました。
- FiltaQuila との連携による自動印刷を実現しました。
- 本文の添付ファイルリストを非表示にするオプションを追加しました。
- 新しいデバッグ設定を追加しました。
- アドレス帳の印刷機能を一時的に取りやめました。

PrintingTools NG で提供される主な拡張機能は次のとおりです。
その他の機能については、設定内容を参照してください。

設定とメニュー
- App メニュー/アドオンとテーマ/拡張機能の管理 から設定にアクセスできます。
- ファイルメニューに [PrintingTools NG 設定] メニューを追加しました。
- ツールバーボタンに [印刷]、[印刷プレビュー]、[PrintingTools NG 設定] を付加しました。

メッセージのレイアウトとフォーマット
- CSS でカスタマイズ可能なヘッダー境界線を追加します。
- ヘッダーの背景色を追加します。
- ユーザー名、フォルダ名、または任意のタイトルを追加します。
- 複数の日時フォーマットが選べます。
- 標準または拡張ヘッダーを非表示にします。
- 標準ヘッダーを並び替えます。
- アイコン付き添付ファイルリストをヘッダーに追加します。
- 本文最下部の添付ファイルリストを非表示にします。
- デジタル署名ファイルを表示します。
- インライン添付ファイルを非表示にします。
- フォントファミリーとサイズを設定します。
- 引用テキストの色、フォント、サイズを強調表示します。
- 選択したテキストのみを印刷します。
- 複数メッセージを印刷します。
- FiltaQuila と連携してサイレントで自動印刷します。
　FiltaQuila 3.4 が必要です。（近日公開予定）
　ベータ版があります。
　https://github.com/RealRaven2000/FiltaQuilla/issues/152#issuecomment-1050708632
